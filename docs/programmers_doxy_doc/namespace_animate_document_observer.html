def AnimateDocumentObserver.addObserver ( )

Adds an AnimateDocumentObserver between FreeCAD's document observers safely.

It's prefered to add an AnimateDocumentObserver using this method, because other ways could result in having multiple document observers added to FreeCAD. Having a lot of document observers slows down FreeCAD due to necessity to inform them all about imminent changes and so on.
