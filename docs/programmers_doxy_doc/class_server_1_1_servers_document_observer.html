Detailed Description

Server's Document Observer to detect a document is closing.

ServersDocumentObserver is a DocumentObserver which will notify the ServerProxy sp when the target document target is being closed. The sp may need this to free a Port it's using, so it can be used in other documents.

To make ServersDocumentObserver active do:

observer = ServersDocumentObserver(server_proxy,
target=FreeCAD.ActiveDocument)
FreeCAD.addDocumentObserver(observer)
