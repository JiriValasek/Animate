Animate is a free and open-source workbench for FreeCAD. It has four tools - Trajectory, Server, Collision Detenction and Control.
Those tools allow you to:

  • Move a set of objects along predefind trajectory
  • Run a TCP server which will execute received messages as FreeCAD commands
  • Detect colisions between multiple objects or sets of objects
  • Control all this through a control panel, record animations and export them

This website is here to help you install Animate into your FreeCAD, get familiar with the tools available and make code changes if required.
